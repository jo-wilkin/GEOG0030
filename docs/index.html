Course Introduction

Welcome to this year’s Geocomputation module, a course that introduces you to both the principles of spatial analysis and the use of programming for data analysis.

Over the next ten weeks, you’ll learn about the theory, methods and tools of spatial analysis whilst implementing small research projects, first using Q-GIS, and then using the R programming language within R-Studio. You’ll learn how to find, manage and clean spatial, demographic and socio-economic datasets, and then analyse them using core spatial and statistical analysis techniques. The course is an excellent precursor for those of you interested in a career in (spatial) data science!

The course will consist of approximately 10 lectures, 10 self-led practicals and 5 seminars (held online), further details of which are provided on the next page, Course Information.

Using this Handbook

All course content, including lectures and practical material, will be contained within this Handbook. As outlined above, you will also need to be part of our Geocomputation Team to have access to the lectures within the handbook.

Each week’s content will be uploaded for Tuesday 10am UK time after the third seminar or coding help sessions.

Key things to look out for in the handbook are Assignments, which are short optional submissions, Practical Instructions, Key Readings and Points of Information.
